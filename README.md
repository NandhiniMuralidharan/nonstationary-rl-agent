# Nonstationary Reinforcement Learning Agent

A modular PyTorch implementation of a Deep Q-Network (DQN) agent designed to evaluate stability and recovery across shifting environment dynamics.

## Key Features
* **Modular Engine**: Decoupled agent and buffer logic for reproducible experimentation.
* **Robustness Benchmarking**: Evaluation of Distribution Shifts, Reward Shaping, and Nonstationary Physics.
* **Comprehensive Analysis**: Full empirical study and mathematical derivations included in the technical report.

## Getting Started
1. Install dependencies:
```bash
   pip install gymnasium[classic_control] torch matplotlib numpy
```
2. Run Experiments
Each experiment is designed as a standalone script that utilizes the modular classes in `engine.py`. Executing these scripts will generate high-resolution (300 DPI) training curves in the `/results` directory:

```bash
  # 1. Mass-Shift Distribution Experiment
  # Evaluates recovery time after a 5x increase in pole length/mass at Episode 200.
  python exp_dist_shift.py

  # 2. Reward Shaping Comparison
  # Contrasts pure sparse signals against potential-based shaping to observe policy collapse.
  python exp_reward_shaping.py

  # 3. Dynamic Gravity Nonstationarity
  # Benchmarks agent stability in an environment where gravity increases by 0.1 m/s² per episode.
  python exp_nonstationarity.py
```
## Experimental Findings

Based on the empirical data generated by this agent, the following behaviors were documented:

* **Resilience to Distribution Shift**: The agent demonstrated high robustness to physical perturbations. While the 5x increase in mass caused an immediate performance drop to near-zero rewards, the target-network stabilization prevented catastrophic forgetting, allowing the agent to return to baseline performance (~300+ score) within 100 episodes.
* **Reward Misspecification & Policy Collapse**: Comparative results showed that while **Shaped Rewards** accelerated convergence in the first 75 episodes, they introduced significant instability in the long term. The agent began over-optimizing for the shaping penalties rather than the terminal goal, settling for a sub-optimal policy compared to the **Sparse Reward** baseline.
* **Limits of Nonstationarity**: In the dynamic gravity experiment, the agent successfully adapted until gravity exceeded approximately **17.0 m/s²**. Beyond this threshold, the rate of environmental change outpaced the update frequency of the Replay Buffer. This caused the buffer to be filled with "stale" low-gravity experiences, leading to a steady decay in convergence stability.
